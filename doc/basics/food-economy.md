# The Settlers II — Buildings: Food economy (summary)

Food keeps **miners** running. Beer is needed to **recruit soldiers** (fresh recruits). :contentReference[oaicite:1]{index=1}

## At-a-glance: what each building is for

- **Fisher** → fish (finite resource)
- **Hunter** → ham (from wildlife; can be boosted with forest edges)
- **Well** → fresh water (supports several buildings)
- **Farm** → grain (space-hungry; needs fertile terrain)
- **Mill** → flour (from grain; feeds bakery)
- **Bakery** → bread (from flour + water; staple mine food)
- **Pig farm** → pigs/meat (grain + water; usually not worth it)
- **Slaughterhouse** → ham (converts pigs/meat to ham)
- **Brewery** → beer (grain + water; for soldier recruitment)

(These are grouped as “Food economy” in the guide.) :contentReference[oaicite:2]{index=2}

## Fisher

### What it does
Produces **fish** by collecting from nearby fish tiles. :contentReference[oaicite:3]{index=3}

### Build / worker / production (from guide)
- Type: hut
- Build cost: **2 boards**
- Worker tool: **rod and line**
- Range: **7**
- Input: fish (from nature)
- Output: **fish** :contentReference[oaicite:4]{index=4}

### Practical notes
- Fish can **run out permanently** on a map; once depleted, it’s gone. :contentReference[oaicite:5]{index=5}
- If a fisher sits at 0% without a depletion message, it may be blocked by a coastal granite deposit or the map might have generated unreachable fish. :contentReference[oaicite:6]{index=6}

## Hunter

### What it does
Produces **ham** by collecting from wildlife. :contentReference[oaicite:7]{index=7}

### Build / worker / production (from guide)
- Type: hut
- Build cost: **2 boards**
- Worker tool: **bow**
- Range: **25**
- Input: animals (from nature)
- Output: **ham** :contentReference[oaicite:8]{index=8}

### Practical notes
- Wildlife tends to appear in forests outside your owned territory; a common trick is to use **foresters** on otherwise “edge” land to create/maintain forest and keep hunters productive (results depend on map; RttR is more likely to generate animals). :contentReference[oaicite:9]{index=9}

## Well

### What it does
Produces **fresh water**. :contentReference[oaicite:10]{index=10}

### Build / worker / production (from guide)
- Type: hut
- Build cost: **2 boards**
- Output: **fresh water** :contentReference[oaicite:11]{index=11}

### Practical notes
- Rule of thumb: plan water for each **bakery**, **brewery**, **donkey breeder**, and **pig farm**, or keep wells near a **storehouse** so distribution stays smooth. :contentReference[oaicite:12]{index=12}

---

## Farm

### What it does
Produces **grain** (the backbone input for bakeries, breweries, etc.). :contentReference[oaicite:13]{index=13}

### Build / worker / production (from guide)
- Type: castle
- Build cost: **3 boards + 3 stones**
- Worker tool: **scythe**
- Range: (fields around the farm)
- Output: **grain** :contentReference[oaicite:14]{index=14}

### Practical notes
- Farms are **space-hungry** and shouldn’t be packed too close.
- They require **fertile terrain** (not sand, mountains, snow, or lava).
- An ideal farm can support up to **6 fields**; **4–5 fields** is typically enough for full productivity, but you’ll often need multiple farms to feed all grain-dependent chains. :contentReference[oaicite:15]{index=15}

## Mill

### What it does
Turns **grain → flour**. :contentReference[oaicite:16]{index=16}

### Build / worker / production (from guide)
- Type: house
- Build cost: **2 boards + 2 stones**
- Input: **grain**
- Output: **flour** :contentReference[oaicite:17]{index=17}

### Practical notes
- The guide suggests matching mills to bakeries: **roughly 1 mill per bakery**, sometimes with an extra mill to keep things at 100% (assuming enough grain). :contentReference[oaicite:18]{index=18}

## Bakery

### What it does
Turns **flour + fresh water → bread** (a key mine food). :contentReference[oaicite:19]{index=19}

### Build / worker / production (from guide)
- Type: house
- Build cost: **2 boards + 2 stones**
- Worker tool: **rolling pin**
- Inputs: **flour**, **fresh water**
- Output: **bread** :contentReference[oaicite:20]{index=20}

### Practical notes
- Place bakeries near **mines** or a **storehouse** to keep miners supplied reliably. :contentReference[oaicite:21]{index=21}

---

## Pig farm

### What it does
Produces **pigs/meat** (consumes grain + water). :contentReference[oaicite:22]{index=22}

### Build / worker / production (from guide)
- Type: castle
- Build cost: **3 boards + 3 stones**
- Inputs: **grain**, **fresh water**
- Output: **pig (meat)** :contentReference[oaicite:23]{index=23}

### Practical notes
- The guide calls out a common downside: pig farms compete for the same grain/water as the bread chain while also taking a lot of space; bread is usually the **more space-efficient** way to feed mines.
- Overall: pig farms are typically optional unless you have a specific reason to run the pig→ham chain. :contentReference[oaicite:24]{index=24}

## Slaughterhouse

### What it does
Turns **pig (meat) → ham**. :contentReference[oaicite:25]{index=25}

### Build / worker / production (from guide)
- Type: house
- Build cost: **2 boards + 2 stones**

- Output: **ham** :contentReference[oaicite:26]{index=26}

### Practical notes
- The guide’s stance: you can maintain mines at 100% without pigs; other food sources are often more efficient, and the hunter/forest approach can also provide strategic benefits. :contentReference[oaicite:27]{index=27}

## Brewery

### What it does
Turns **grain + fresh water → beer** (used for recruiting soldiers). :contentReference[oaicite:28]{index=28}

### Build / worker / production (from guide)
- Type: house
- Build cost: **2 boards + 2 stones**
- Inputs: **grain**, **fresh water**
- Output: **beer** :contentReference[oaicite:29]{index=29}

### Practical notes
- Place near an **armory** or a **storehouse** so beer reaches where it’s needed. :contentReference[oaicite:30]{index=30}

## Quick placement heuristics (condensed)

- Prefer the **bread chain** (farm → mill → bakery) as the “default” scalable mine food solution. :contentReference[oaicite:31]{index=31}
- Use **fishers** early when fish is nearby, but remember it’s finite. :contentReference[oaicite:32]{index=32}
- Use **hunters** when you can create/keep forests on borders/edge land (often with foresters) to stabilize wildlife availability. :contentReference[oaicite:33]{index=33}
- Always plan **wells** alongside water-consuming buildings (bakery, brewery, donkey breeder, pig farm) or near a storehouse for distribution. :contentReference[oaicite:34]{index=34}
