A. Note de référence

Le jeu nécessite une carte graphique compatible OpenGL 2.0 avec au moins 64 Mo de mémoire.
Un CPU de 800 MHz est suffisant.

Vous aurez également besoin d’une version installée de "The Settlers 2 Gold Edition"
ou de la version originale + CD Missions.

--------------------------------------------------------------------------------

B. Installation

Windows, Linux et Mac sont supportés.

Copiez les dossiers DATA et GFX du jeu original dans le dossier du jeu RTTR.

Le jeu ne modifie pas les fichiers originaux.

--------------------------------------------------------------------------------

C. Jeu

Modes disponibles :
- Solo
- Multijoueur

Multijoueur :
- IP directe
- LAN
- Lobby

Port utilisé : 3665 TCP

Replays :
Permettent de revoir une partie. Utilisez + et - pour la vitesse.

Sauvegardes :
Fonctionnent comme dans le jeu original.
